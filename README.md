## Setup Backend
1. cd backend <br />
2. pip install -r requirements.txt <br />
3. python manage.py createsuperuser <br />
4. python manage.py runserver <br />

## Setup Frontend
1. cd frontend <br />
2. npm install --save express <br />
3. npm start <br />
